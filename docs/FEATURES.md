# Core Features
## Layout
### Top bar
- Show the hierarchy of pages (current path) -> Similar to file system
- Side bar
    - Home
    - Show ALL root pages -> name and ID will do
    - Settings

## Pages
### Home page
- Show recents (most recent 5)
- Add new page
    - Actuall creates the new page
    - Redirects to new page

### Individual Page:
- Show title
- Show and editor: realtime edit


## Components:
- Top bar -> for current page parents and children
- Side bar
    - Home
    - ALL root pages
    - Settings

- Editor (used for individual page)
    - support / commands such as:
        - /1, /2, /3
        - - -> for bullet points
        - 1. 2. -> for numbered points
    - Embed links (copy and paste)
        - mainly for videos
    - Embedded videos
        - Allow AI summary
        - Allow taking of timestamps and snapshot
    - Allow users to add images
